INTRODUCTION
------------

The Catalog Application is a catalog of items in categories. This application can Create, Read, Edit, and Delete categories and items. Users have access to these features by signing in using a Google Sign-in. 

REQUIREMENTS
------------

In order to run this application, you'll need to install Vagrant and VirtualBox. Then, you have to clone the fullstack-nanodegree-vm repository. Follow the instructions found here - https://docs.google.com/document/d/1jFjlq_f-hJoAZP8dYuo5H3xY62kGyziQmiv9EPIA7tM/pub?embedded=true - for more information on setting that up.

INSTALLATION
------------

To install this application, fork this repository to your local computer. Then, run the application.py file. To access the application in your browser, go to http://localhost:5000. 

CONFIGURATION
-------------

In order to create, edit, and delete categories and items, you must have a Google account. If you don't, you can register for one at Google.
